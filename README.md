# BG Drinov kbd
 Historic Bulgarian (typewriter) keyboard layout 
 
The layout is created by Microsoft Keyboard Layout Creator 1.4  
![BGDrinov](https://user-images.githubusercontent.com/1756473/152694239-9169d422-3044-421b-b6f6-43845e872258.jpg)
![BGDrinovCaps](https://user-images.githubusercontent.com/1756473/152694236-e0d30732-6f6a-4172-a5e1-55750a72cf27.jpg)
![BGDrinovShft](https://user-images.githubusercontent.com/1756473/152694238-c8047602-fe3e-4f9d-9cd3-7526e2563dbe.jpg)

Other supported alphabets are Serbian, Maccedonain and Russian via deadkays (orange)
